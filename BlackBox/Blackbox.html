Basic Layout Concepts

Block Elements

Block elements take up the full width available and stack vertically. Examples include div, h1 to h6, p, ul, ol, and form elements.

Inline Elements

Inline elements only take up the space needed for their content and don't stack vertically. Examples include span, a, img, b, i, and em elements.

Containing Elements

Containing elements help group and position related content. A div is a common containing element, but any block or inline element can serve as a containing element.

Display Property

The display property can change an element's layout behavior. For example, setting display: block on an inline element will convert it to a block element. Similarly, setting display: inline on a block element will convert it to an inline element.
